# Changelog

## Unreleased

- Added deterministic LuigiOS boot, Game Mode, desktop wallpaper, mascot, settings,
  and recovery assets with exact provenance and OLED-black surface checks.
- Added transactional demo branding activation, backup, rollback, and watchdog tools.
- Made COSMIC the sole shipping desktop target; the live Plasma prototype is frozen as
  historical compatibility evidence.
- Excluded generated `dist/` evidence and release artifacts from source graphics and
  repository hygiene scans.
- Added original controller-first interface guidelines with clean-room visual rules and
  hard offline/no-account qualification for the vanilla LuigiOS distribution.
- Added an experimental Developer Mode SSH connectivity guard that monitors route health
  and reconnects the last healthy ConnMan Wi-Fi service without restarting networking.
- Added a secure implementation contract with memory-safe Rust defaults, native binary
  hardening, least-privilege runtime rules, dependency audits, sanitizers, and fuzzing.

## 0.1.0-dev.1

First public-development source milestone.

- Steam Deck Batocera prototype lifecycle, controller, storage, and recovery scripts.
- Pinned Batocera Unofficial Add-Ons package integration.
- Private firmware/config/save and separate ROM-library migration profiles and tooling.
- TUF-authorized P2P distribution policy with upstream Batocera swarm federation.
- Trusted Vault and per-system library sharing design contracts.
- Btrfs Restore Points proposal.
- Side-by-side Batocera Buildroot SDK with Docker/Podman support.
- Governance, DCO, credits automation, issue forms, CI, and release/test procedures.
- Pinned COSMIC desktop-image contract, fullscreen host session, controller bridge,
  Flatpak preflight, migration plan, and recovery integration.
- Original LuigiOS marks, semantic icons, boot stills, desktop and community artwork,
  deterministic renderer, provenance, and graphics compliance gate.
- OLED-black dominant surfaces and wallpaper shadows with charcoal elevation
  retained for readable handheld and television navigation.
- Paired angel and demon variants of the same caretaker raccoon, with packaged
  UI assets and explicit safe-path, expert-risk, and accessibility semantics.
- Gamer Mode and Developer Mode product contract for a graphical, controller-first
  default with reversible access to development tools and Linux internals.
- Steam Deck reference-platform policy with non-blocking community, downstream,
  experimental, and unverified tiers for additional hardware.
- Unified LuigiOS naming, interface voice, and player-ownership contract: games first,
  local-first operation, portable data, optional services, and no paid default placement.
- Discord and Reddit launch blueprints, moderation policy, community onboarding,
  and an open downstream/OEM compatibility program.
- Experimental-to-beta qualification policy, read-only SSH device QA, and a staged
  fresh-install process that physically protects the existing ROM SD card.

This milestone is source and architecture for contributors, not an end-user OS image.
