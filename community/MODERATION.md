# Community Moderation

## Principles

Moderation protects participation rather than winning arguments. Apply the
smallest intervention that stops harm, explain the relevant rule, and keep a
private record for actions stronger than a routine content edit. Never ask for
passwords, authentication codes, private firmware, keys, saves, or ROM files.

## Response ladder

1. Clarify or redirect an honest mistake.
2. Edit, label, slow, lock, or remove content that creates immediate risk.
3. Warn the member with the rule, evidence, and expected correction.
4. Apply a temporary timeout or ban for repeated or substantial disruption.
5. Permanently ban credible threats, doxxing, severe harassment, malware, or
   deliberate credential and proprietary-payload trafficking.

Serious harm can skip steps. Disagreement, criticism, or competitor discussion is
not misconduct by itself.

## Records and appeals

Record date, platform, moderator, rule, evidence link, action, duration, and any
appeal outcome in a private moderator log. Do not copy sensitive content into the
log when a hash or restricted link is enough. A moderator named in a complaint
must not decide its appeal. Review permanent bans after 90 days when requested,
unless safety or legal obligations prevent contact.

## Routing

| Report | Route |
| --- | --- |
| Reproducible software bug | GitHub issue form |
| Security vulnerability | Private GitHub Security Advisory |
| Setup question | GitHub Discussion or Discord help forum |
| Conduct or privacy concern | Platform modmail/moderator ticket |
| Copyright or trademark notice | Project maintainer, privately |
| Immediate physical danger | Platform safety tools and local emergency services |

Moderators follow the host platform's rules in addition to the LuigiOS Community
Code. Reddit moderators should review the current [Moderator Code of Conduct](https://support.reddithelp.com/hc/en-us/articles/15484183932564-Get-to-know-Reddit-s-Moderator-Code-of-Conduct),
and Discord moderators should keep AutoMod alerts restricted to the staff team.
