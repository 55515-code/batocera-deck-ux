# Welcome to LuigiOS

LuigiOS is an early, community-built, controller-first gaming OS project. The goal
is straightforward: games first, your device, your choices.

Today this is a contributor source preview, not a finished consumer image. Expect
active development and honest limitations. Project decisions, code, release
evidence, and reproducible bug reports live on GitHub so useful knowledge remains
searchable and reviewable.

You are welcome here whether you write code, test hardware, make art, improve
documentation, package software, repair devices, preserve games, or simply want a
better handheld experience. Share what you tried, what happened, and what would
make the project easier to use.

LuigiOS does not distribute games, ROMs, BIOS files, console firmware, keys, or
other proprietary payloads. Please read the community rules before posting.
