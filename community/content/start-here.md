# Start here

LuigiOS is currently an experimental contributor preview. There is not yet a
general-purpose, supported installer image. Do not rely on it for irreplaceable
data, and keep tested backups and recovery media before changing a device.

Good ways to help right now:

- Review an open issue or a focused design proposal on GitHub.
- Test a documented path on hardware you physically possess and report exact results.
- Improve setup, recovery, accessibility, or localization documentation.
- Share original art or user-experience feedback with source and license details.
- Ask a clear question in Help; move confirmed bugs to GitHub with reproduction steps.

Include the device model, LuigiOS revision, expected result, actual result, and any
carefully redacted logs. Never upload credentials, private firmware, keys, saves,
ROMs, or personally identifying data.
