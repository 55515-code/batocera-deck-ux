# Support

Use GitHub Discussions for setup questions and design conversation. Use issue forms for
reproducible bugs, hardware test reports, and concrete feature proposals. Third-party
add-ons remain supported by their own upstream communities unless the failure is in this
project's packaging or lifecycle integration.

Steam Deck is the core reference platform. Reports for other hardware are welcome and
may be listed as community-tested, downstream-maintained, experimental, or unverified.
Those listings are evidence, not a LuigiOS support promise. Device vendors and named
downstream maintainers remain the support contact for the profiles they own. See
[ADR 0007](docs/adr/0007-reference-platform-and-community-hardware.md).

The Discord community is in a controlled soft launch at
<https://discord.gg/BDfCJPUeVG>. Reddit setup remains blocked by Reddit. Durable
bugs, security reports, release evidence, and project decisions still belong on
GitHub; see the versioned [community configuration](community/README.md).

Before reporting a bug, test a clean profile when possible and attach the diagnostics
bundle produced by the future graphical Diagnostics screen. Until that exists, redact
logs carefully and never upload credentials, private firmware, BIOS, keys, saves, or ROMs.
